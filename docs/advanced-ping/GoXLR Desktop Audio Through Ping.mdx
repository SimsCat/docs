# Send GoXLR Desktop Audio through Ping

- In Ping, make sure you’re using the “Chat Mic” output from GoXLR

<img src="https://lh3.googleusercontent.com/Y6njdPTfyGTtyuC5vrbAfuhyXvQb7RTACc1AeukvyikO_C0-qx6RB_X-ZbO8Ym_tPCJFxSwKol1cUsAR0uSIYrDLAqdwaAjq4ZSHCruGMEBf7ROFHLGSi6RNQetwRezc5sNtsGYhfdTPsrFr6Q"></img>

- In the GoXLR Routing Table, make sure that System Audio is ticked on in the Chat Mic Section

![https://lh3.googleusercontent.com/mEfYFT__Iz7SgMpr8TPIQGFFrRkKpg16j7Xvm1a4O78PYAo9pTjtznijAWj_EztNtF6SWT57waIh1or-O85x9S3z-AQntrareJlsVyRUD5LHpDILIsf_yoNAJQ-g7-dvzU_0D7i6rDrTgrBn_w](https://lh3.googleusercontent.com/mEfYFT__Iz7SgMpr8TPIQGFFrRkKpg16j7Xvm1a4O78PYAo9pTjtznijAWj_EztNtF6SWT57waIh1or-O85x9S3z-AQntrareJlsVyRUD5LHpDILIsf_yoNAJQ-g7-dvzU_0D7i6rDrTgrBn_w)

- In Ping, go to your dashboard and click on options under your room

![https://lh5.googleusercontent.com/pNOs4ovss9TwNUti4RM7lhqGbSxOq53Gupoi83sOXqEtK7Eu2YV9b8-i9J2SCKLtApk9m-b5AFLL1fYzVoO_Xav5BNX_uo1fPETmUrNGTChi-uMdmA4N4acExEdYMcddsAyQk3d5za5hZczIHA](https://lh5.googleusercontent.com/pNOs4ovss9TwNUti4RM7lhqGbSxOq53Gupoi83sOXqEtK7Eu2YV9b8-i9J2SCKLtApk9m-b5AFLL1fYzVoO_Xav5BNX_uo1fPETmUrNGTChi-uMdmA4N4acExEdYMcddsAyQk3d5za5hZczIHA)

- Then on the left side, go to Experimental

![https://lh3.googleusercontent.com/5zAcck43_RXv8LEv7_Ap8hAnobafhh_qJaChdGShywuQemOz9QXIKfsIpeIR5-tuVXjmqH1CY6zX0lYWPS6CuWAG9WoCZ3ZvDfIolBz-LB9VvBMjQzAA0E3HzSryCnwrzDbY-tWYyRWE1_5yrg](https://lh3.googleusercontent.com/5zAcck43_RXv8LEv7_Ap8hAnobafhh_qJaChdGShywuQemOz9QXIKfsIpeIR5-tuVXjmqH1CY6zX0lYWPS6CuWAG9WoCZ3ZvDfIolBz-LB9VvBMjQzAA0E3HzSryCnwrzDbY-tWYyRWE1_5yrg)

- And make sure to turn off Automatic Gain Control, Automatic Noise Supression, and Automatic Echo Cancellation.

![https://lh4.googleusercontent.com/Yug1QqLPH8TfR6MPHZbjyyIPzrBChL2TbfX-MscL9wDFGpWh6H_j9R6exA-JWtBU_zyir6yyIn-Wh7lIwaxH37iYi_cw00943QnEQBx0Shl-fNa60s55wumbQNmdCg9AIic0eGxg1j5O13Btmw](https://lh4.googleusercontent.com/Yug1QqLPH8TfR6MPHZbjyyIPzrBChL2TbfX-MscL9wDFGpWh6H_j9R6exA-JWtBU_zyir6yyIn-Wh7lIwaxH37iYi_cw00943QnEQBx0Shl-fNa60s55wumbQNmdCg9AIic0eGxg1j5O13Btmw)

- Side note, if your guests can hear themselves in the playback, make sure that whatever you’re using to hear them (Discord or Google Chrome) is being sent to “Chat” in Windows
  - You can get there by Right Clicking the Volume Icon in the bottom right of the system tray, then going to Sound Settings, then going to “App volume and device preferences”
